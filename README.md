# 智能窗帘光温控制系统

基于 STM32 + ESP8266 + OneNET + 微信小程序的智能窗帘控制系统毕业设计。

## 功能特性

### 环境感知
- 温湿度采集 (DHT11)
- 光照强度检测 (光敏电阻 + ADC)

### 自动控制
- 基于温度和光照的智能窗帘调节
- 支持自动/手动模式切换
- 可调温度阈值 (-20~60°C)
- 可调光照阈值 (0~1000 lx)

### 远程控制
- 微信小程序实时数据查看
- 远程窗帘开合控制
- 远程参数设置

### 状态显示
- OLED 多页面显示
- 温湿度、光照、窗帘状态
- 云连接状态指示

## 硬件清单

| 模块 | 型号/规格 | 数量 | 用途 |
|------|-----------|------|------|
| 主控芯片 | STM32F103C8T6 | 1 | 系统核心控制器 |
| 温湿度传感器 | DHT11 | 1 | 采集环境温湿度 |
| 光敏电阻 | 5516 | 1 | 采集光照强度 |
| OLED 显示屏 | 0.96 寸 I2C | 1 | 显示系统状态 |
| 步进电机 | 28BYJ-48 | 1 | 驱动窗帘开合 |
| 电机驱动 | ULN2003 | 1 | 驱动步进电机 |
| WiFi 模块 | ESP8266 (ESP-01S) | 1 | 连接云平台 |
| LED 指示灯 | 3mm | 3 | 报警/状态提示 |
| 按键 | 轻触开关 | 4 | 本地操作 |

### 引脚分配

| 模块 | 引脚 | 功能 |
|------|------|------|
| DHT11 | PA6 | 数据线 |
| 光敏电阻 | PA4 (ADC1_CH4) | 模拟输入 |
| OLED | PB8 (SCL), PB9 (SDA) | I2C 通信 |
| 步进电机 | PA8~PA11 | 电机控制 |
| ESP8266 | PA2 (TX), PA3 (RX) | USART2 |
| LED | PB5~PB7 | 报警指示 |
| 按键 | PC13~PC15, PA0 | 用户输入 |

> **注意**: 调试串口 USART1 (PA9/PA10) 与步进电机引脚存在复用冲突，如需调试请禁用电机功能。

## 软件组成

### STM32 固件

基于 Keil MDK 开发，主要模块：

| 模块 | 文件 | 功能 |
|------|------|------|
| 主程序 | `智能窗帘/user/main.c` | 系统初始化和任务调度 |
| WiFi 驱动 | `智能窗帘/API/ESP8266/` | ESP8266 AT 指令和 MQTT 通信 |
| 按键控制 | `智能窗帘/API/key/` | 按键扫描和自动控制逻辑 |
| 电机控制 | `智能窗帘/API/motor/` | 步进电机开合控制 |
| 传感器 | `智能窗帘/API/DHT11/`, `LDR/` | 温湿度和光照采集 |
| 显示驱动 | `智能窗帘/API/oled/` | OLED 显示和页面管理 |

### 微信小程序

基于微信小程序框架开发，包含 3 个页面：

| 页面 | 功能 |
|------|------|
| 首页 | 实时数据看板、模式切换 |
| 窗帘页 | 窗帘开合控制、操作记录 |
| 设置页 | 温度/光照阈值调整 |

## OneNET 物模型属性

| 属性标识符 | 类型 | 方向 | 说明 |
|------------|------|------|------|
| `Tmp` | float | 上报 | 温度 (°C) |
| `Hum` | float | 上报 | 湿度 (%) |
| `ldr` | int | 上报 | 光照强度 (lx) |
| `curtain` | int | 上报/下发 | 窗帘关闭度 (0=全开, 100=全关) |
| `mode` | int | 上报/下发 | 运行模式 (0=手动, 1=自动) |
| `tem_set` | float | 上报/下发 | 温度阈值 (-20~60) |
| `ldr_set` | int | 上报/下发 | 光照阈值 (0~1000) |
| `Tem_beep` | bool | 上报 | 温度报警标志 |
| `Hum_beep` | bool | 上报 | 湿度报警标志 |
| `CO2_beep` | bool | 上报 | 光照报警标志 |

## 快速开始

### 硬件准备

1. 按照电路图连接各模块
2. 确保 STM32、ESP8266、电机驱动供电正常
3. 烧录固件到 STM32

### 云平台配置

1. 注册 OneNET 账号: https://open.iot.10086.cn/
2. 创建产品和设备
3. 配置物模型属性（参照上方属性表）
4. 获取设备 ID 和 API Key

### 小程序配置

1. 使用微信开发者工具打开 `智能窗帘小程序/` 目录
2. 编辑 `utils/config.js`，填入 OneNET 配置：
   ```javascript
   module.exports = {
     USER_ID: '你的用户ID',
     ACCESS_KEY: '你的access_key',
     // ...
   }
   ```
3. 在微信公众平台配置合法域名: `https://iot-api.heclouds.com`

### 按键操作

| 按键 | 功能 |
|------|------|
| KEY1 | 切换设置项（温度阈值/光照阈值） |
| KEY2 | 减小阈值 / 手动关窗 |
| KEY3 | 增大阈值 / 手动开窗 |
| KEY4 短按 | 切换显示页面 |
| KEY4 长按 | 切换自动/手动模式 |

## 自动控制逻辑

系统在自动模式下根据温度和光照组合执行不同策略：

| 温度 | 光照 | 动作 | 原因 |
|------|------|------|------|
| - | 很低 | 关窗 | 保护隐私 |
| 高 | 强 | 关窗 | 防晒隔热 |
| 高 | 弱 | 开窗 | 通风散热 |
| 低 | 强 | 开窗 | 采光采暖 |
| 低 | 弱 | 关窗 | 保温节能 |
